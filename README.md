# Point of Sale System (POS) para una tienda de abarrotes
Sistema de punto de venta desarrollado con:
Frontend: React + Vite
Backend: Node.js + Express
Base de datos: MySQL
Este proyecto permite gestionar productos y visualizar información desde un panel de control.

# Estructura del proyecto

POS,
POSFrontend - Aplicación React, 
POSBackend - API y lógica del servidor, 
DataBase db.sql - Script de la base de datos

# Requisitos

Antes de ejecutar el proyecto debes tener instalado:
Node.js
MySQL

# Configuración de la base de datos
1. Abrir MySQL.
2. Importar el archivo: DataBase/db.sql
Esto creará las tablas necesarias para el sistema.

# Ejecutar el Backend
Ir a la carpeta del backend: cd POSBackend
Instalar dependencias: npm install
Ejecutar el servidor: npm run dev
El backend iniciará en http://localhost:3001

# Ejecutar el Frontend
Ir a la carpeta del frontend: cd POSFrontend
Instalar dependencias: npm install
Ejecutar la aplicación: npm run dev
El frontend se ejecutará en http://localhost:5173/

# Tecnologías utilizadas
* React
